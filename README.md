# To-do List App
A simple to-do list application built with HTML, CSS, and vanilla JavaScript.

## 📸 Preview
![To-do List Preview](./img/todo-list_2.gif)

## ✨ Features
- Add tasks
- Delete individual tasks
- Clear all tasks at once
- Strike-through completed tasks
- Local storage support (tasks persist after page reloads)
- Fully responsive

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## 🔮 Future Updates
- UI refinement (spacing, colors and layout)
- Responsive design improvements for better mobile experience
